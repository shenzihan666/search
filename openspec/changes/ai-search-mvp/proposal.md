# AI Search MVP - Phase 1

## Why

构建一个最小可用的 AI 搜索框，验证核心交互模式：快捷键唤出 + AI 对话。

## What Changes

MVP 阶段核心功能：
- **全局快捷键**: `Alt+Space` 唤出/隐藏
- **单模型查询**: 先支持 OpenAI API
- **系统托盘**: 后台常驻
- **基础 UI**: 输入框 + 结果显示

## Capabilities

### New Capabilities

- `global-hotkey`: 全局快捷键唤出（简化版，先不支持自定义）
- `single-model-query`: 单模型查询（OpenAI）
- `system-tray`: 系统托盘常驻
- `basic-ui`: 基础界面（输入框 + 结果区）

### Modified Capabilities

(无)

## Impact

- Phase 1 of 3
- 目标：可发布的最小可用版本
- 后续阶段在此基础上扩展多模型对比功能
